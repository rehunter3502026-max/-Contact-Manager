# -Contact-Manager

A simple command-line Contact Management System built in Python that allows users to store, search, view, and delete contacts dynamically.

This project demonstrates fundamental Python concepts including dictionaries, functions, loops, and conditional logic.

🚀 Features

Add new contacts

View all saved contacts

Search contact by name

Delete existing contact

Menu-driven interactive interface

🧠 Concepts Used

Python dictionaries

Functions

While loops

Conditional statements

Basic input/output handling

💡 Learning Outcomes

Managing data using dictionaries

Building modular programs using functions

Creating interactive CLI applications

Understanding CRUD operations (Create, Read, Update, Delete)
